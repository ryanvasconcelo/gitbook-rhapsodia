# Jornada do Usuário - Rhapsodia

## Dia 1: Primeiro Acesso

### Manhã - Onboarding (30 minutos)

**08:00 - Login Inicial**
1. Receba as credenciais do administrador
2. Acesse https://rhapsodia.suaempresa.com
3. Faça login com email corporativo e senha

**08:15 - Tour Guiado**
1. Complete o tour interativo de 5 minutos
2. Familiarize-se com as 3 áreas principais:
   - Barra lateral: Histórico e workspaces
   - Área central: Chat principal
   - Barra superior: Seleção de modelos

**08:30 - Primeira Conversa**
Teste básico - Digite no chat:
"Olá! Sou novo na plataforma. O que você pode fazer por mim?"

### Tarde - Primeiras Tarefas Reais (2 horas)

**14:00 - Análise de Documento**
1. Clique no ícone de clip 📎
2. Faça upload de um relatório ou planilha
3. Pergunte: "Analise este documento e destaque os pontos principais"
4. Salve a resposta clicando nos três pontos > Exportar > PDF

**15:00 - Email Profissional**
Prompt sugerido:
"Escreva um email profissional para [cliente] sobre [assunto]. Tom: formal, mas cordial. Máximo 200 palavras."

**16:00 - Resumo de Reunião**
1. Cole suas anotações no chat
2. Use o prompt: "Transforme estas anotações em uma ata formal de reunião com: participantes, tópicos discutidos, decisões tomadas e próximos passos"

## Dia 2-5: Construindo Proficiência

### Explorando Modelos Diferentes

**GPT-4o** (Padrão)
- Melhor para: Tarefas gerais, redação, análise
- Velocidade: Rápida
- Custo: Médio

**Claude 4.1 Sonnet**
- Melhor para: Análises complexas, código
- Velocidade: Moderada
- Custo: Alto

**Gemini 3.0 Pro**
- Melhor para: Documentos longos (até 1M tokens)
- Velocidade: Rápida
- Custo: Baixo

### Criando seu Primeiro Workspace

1. Clique em "+" na seção Workspaces
2. Nome sugestão: "Projeto [Nome]" ou "[Departamento] - [Mês]"
3. Adicione colegas pelo email
4. Defina permissões (visualizar/editar)

### Usando Templates de Prompts

Navegue até a biblioteca em Admin Panel > Prompts:
- **Financeiro**: "Analise esta DRE e sugira melhorias"
- **RH**: "Crie uma descrição de vaga baseada nestes requisitos"
- **Vendas**: "Gere um follow-up para proposta enviada há X dias"

## Semana 2: Uso Avançado

### Integrando com Base de Conhecimento

1. Acesse Admin Panel > Conhecimento
2. Verifique as coleções disponíveis
3. No chat, use "@conhecimento" para buscar informações internas
4. Exemplo: "@conhecimento qual é nossa política de férias?"

### Criando Automações Simples

**Relatório Semanal Automatizado**
1. Salve o prompt como template
2. Configure recorrência (toda segunda, 9h)
3. Defina destinatários do email
4. Prompt exemplo:
"Gere um relatório executivo com:
KPIs da semana
Principais realizações
Riscos identificados
Prioridades para próxima semana
Formato: bullet points, máximo 1 página"


### Monitorando seu Uso
1. Acesse seu perfil > Uso e Custos
2. Visualize:
   - Tokens consumidos por modelo
   - Custo de tokens no período

## Semana 3-4: Maestria

### Técnicas Avançadas de Prompt
**Chain-of-Thought (Raciocínio em Etapas)**
Analise este problema passo a passo:
Primeiro, identifique os dados relevantes
Depois, calcule as métricas principais
Por fim, forneça recomendações baseadas nos resultados

**Few-Shot Learning (Exemplos)**
Classifique estes tickets de suporte:
Exemplo 1: "Não consigo acessar" -> Categoria: Login
Exemplo 2: "Sistema lento" -> Categoria: Performance

Agora classifique: [seu ticket]

### Colaboração em Equipe
1. Compartilhe conversas úteis com a equipe
2. Crie workspaces por projeto
3. Estabeleça convenções de nomenclatura
4. Use tags para organizar conversas: #financeiro #urgente #relatório

## Dicas de Ouro

1. **Seja específico**: "Resuma em 3 bullets" em vez de "Resuma"
2. **Forneça contexto**: "Como gerente de vendas, preciso..."
3. **Itere**: Refine a resposta com "Agora adicione X" ou "Torne mais Y"
4. **Salve o que funciona**: Crie templates dos prompts bem-sucedidos
5. **Experimente modelos**: Cada um tem forças diferentes, explore
